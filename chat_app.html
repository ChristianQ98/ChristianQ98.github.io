Chat App

  • Built using React.js, HTML, and CSS on the front-end and Express.js, Node.js, and Socket.io on the back-end
  • Implemented web sockets to provide real-time communication between the client and the server, which allows users to send and receive messages in real-time
  • Utilized session to store the user’s entered information which is used to display a home page and welcome message consisting of the user’s information
