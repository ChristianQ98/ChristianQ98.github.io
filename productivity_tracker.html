Productivity Tracker

  • Built using HTML, CSS, Javascript, Python, Flask, MySQL Workbench, and Jinja2
  • Integrates a Register/Login page, which gets stored into the MySQL database and hashes the passwords using Bcrypt
  • Utilizes Regex for additional security and password validations
  • Allows a user to submit a form asking them how their overall day went, how productive they were, and enter any additional comments they have
